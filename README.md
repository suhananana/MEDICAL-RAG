# MEDICAL-RAG
#The Medical RAG (Retrieval-Augmented Generation) System is an AI-powered application designed to provide accurate and context-aware medical information by combining information retrieval with natural language generation.

Instead of relying only on a pre-trained model, this system retrieves relevant medical data from a trusted knowledge base and then generates precise, human-like responses. This improves reliability and reduces hallucinations in critical domains like healthcare.
